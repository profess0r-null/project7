Cybersecurity Subdomains

Cybersecurity is a vast field with many specializations. Here are a few key subdomains:

Network Security

This involves protecting the computer network infrastructure from unauthorized access, misuse, or theft. It includes securing hardware and software. Key areas are firewalls, VPNs, Intrusion Detection Systems (IDS), and anti-virus software.

Application Security (AppSec)

This focuses on finding and fixing vulnerabilities in software and applications. It covers web applications, mobile apps, and APIs. Common practices include secure coding, vulnerability scanning (SAST/DAST), and bug bounty programs.

Penetration Testing (Offensive Security)

Also known as ethical hacking, this is the practice of testing a computer system, network, or web application to find security vulnerabilities that an attacker could exploit. Pentesters simulate real-world attacks to assess security strength.

Incident Response (Defensive Security)

This field deals with the "aftermath" of a security breach. Incident Responders identify the attack, contain the damage, remove the threat, and help the organization recover. It's a critical part of the defensive "Blue Team" operations.

Cloud Security

As more companies move to services like AWS, Azure, and Google Cloud, this field has become crucial. It involves securing cloud environments, data, and applications from threats, focusing on misconfigurations, access control, and compliance.

Cryptography

This is the practice and study of techniques for secure communication in the presence of third parties. It involves creating and analyzing protocols that prevent malicious actors from accessing information, ensuring data confidentiality, integrity, and authenticity.
